# 🍳 Ricettario

A fast, private, and offline-ready digital cookbook that lives entirely in your browser. No servers, no accounts, and no data tracking—just your recipes, organized and ready to cook.

## ✨ Features

- **100% Local Storage**: Your recipes never leave your browser. Your data, your rules.
- **Offline Access**: Works perfectly without an internet connection.
- **PWA Ready**: Install it on your desktop or mobile device for a native-like experience.
- **Intuitive Interface**: Clean, distraction-free design to focus on your cooking.
- **Instant Search**: Find your favorite recipes in milliseconds.
- **Data Portability**: Easily import and export your recipe collection.

## 🚀 How to use

1. **Online**: [Open the live demo](https://gdr-sys.github.io/Ricettario/)
2. **Offline**: 
   - Clone this repository or download the `index.html`.
   - Open the file in any modern web browser.
   - Start adding your recipes!

## 🛠 Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Storage**: IndexedDB & LocalStorage
- **Offline**: Service Workers (PWA)
